# CallableFunction
可以在运行时通过面板激活函数，支持传递多种参数。只需要将需要Invoke的方法加``[CallableFunc]``标签即可

在Tools->CallableFunctionWindow只需要拖入父节点，即可列举所有的CallableFunc函数
